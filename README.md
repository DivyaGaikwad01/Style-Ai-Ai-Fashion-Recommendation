# StyleAI – AI Fashion Recommendation System

## 🚀 Overview
StyleAI is an AI-powered fashion recommendation system that analyzes a user’s skin tone from uploaded images and suggests personalized outfit and color recommendations.

## 🧠 How it Works
1. User uploads an image  
2. OpenCV detects face and analyzes skin tone  
3. AI model processes tone category (fair, light, medium, dark)  
4. Grok API + LLM generates fashion recommendations  
5. Results are displayed instantly on UI  

## 🛠 Tech Stack
- React + TypeScript (Frontend)
- Tailwind CSS
- Python + Flask (Backend)
- OpenCV (Image Processing)
- Grok API / LLM (AI Recommendations)
- Supabase (if used for backend/database)

## 🎯 Features
- Skin tone detection using image processing  
- AI-based outfit recommendations  
- Responsive modern UI  
- Fast real-time results  
- Personalized fashion suggestions  

## 🧩 System Architecture
User → Upload Image → OpenCV Processing → Skin Tone Detection → AI Model (Grok/LLM) → Outfit Recommendations → Display Results
