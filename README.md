# StyleAI – AI Fashion Recommendation System

## 🚀 Overview
StyleAI is an AI-powered fashion recommendation web application generated using Lovable during a hackathon. The project takes user-uploaded images and provides personalized outfit recommendations based on skin tone analysis.

## 🧠 How It Works
- User uploads an image  
- System analyzes facial region using OpenCV-based processing  
- Skin tone is classified into categories (fair, light, medium, dark)  
- AI model (Grok API / LLM) generates fashion recommendations  
- Results are displayed through a responsive UI  

## 🛠 Tech Stack
- React + TypeScript (Frontend)
- Tailwind CSS
- Python + Flask (Backend integration)
- OpenCV (Image Processing)
- Grok API / LLM (AI Recommendations)
- Supabase (if used)
- Lovable (Project generator)

## 🎯 Features
- Image-based skin tone detection  
- AI-generated fashion suggestions  
- Responsive web interface  
- Real-time recommendation display
